## Screenshot Instructions for Pitch Deck

### Recommended capture sequence (20 minutes total):

1. **Open browser in Incognito mode** → joecafebrand.netlify.app
2. **DevTools** → Mobile (iPhone 12 Pro, 390×844)
3. Capture Welcome screen
4. Login as Google → HomeView → capture menu
5. Add Masala Dosa + Filter Coffee → capture cart bar
6. Tap Process Order → capture PaymentView
7. Select Cash → capture confirmation
8. **Second tab**: Login as cashier → capture CashierView with pending order
9. Approve → return to student tab → capture QR
10. **Third tab**: Login as server → navigate to SERVER tab → scan student QR
11. Capture manifest (food image + items)
12. Tap SERVE ALL → capture success state
13. Return to student tab → capture "Order Complete" state
14. Login as admin → capture Dashboard metrics

## Demo Video Specifications (for recorded backup)

- **Resolution**: 1080p (portrait for phone, landscape for desktop console)
- **Duration**: 2 minutes max
- **Voiceover**: English, energetic but professional
- **Format**: MP4 H.264 for compatibility
- **Compression**: < 50MB for easy email sharing
- **Captions**: Yes — accessible for hearing-impaired jury members
